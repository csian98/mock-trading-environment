## Install Dependencies

Run the following bash commands under the project root directory

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run Trading Environments

Run the following bash commands under the project root directory

bollinger rsi strategy:

```bash
python3 main_bollinger_rsi.py
```

ucb strategy:

```bash
python3 main_ucb.py
```

lstm strategy:

```bash
python3 main.py
```
